# FIYA

FIYA Single page company profile website layout built using Tailwindcss.

![Fiya template preview](https://raw.githubusercontent.com/naxeem/fiya-company/master/preview.jpg)


## Installation

```bash
npm install
```

## Build tailwind

```bash
npm run watch
npm run prod
```
